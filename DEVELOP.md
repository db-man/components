# Develop

## Install

```
npm i
```

Dev

```
npm start
```

## Publish

```
npm run release
```

## Preview demos

- http://localhost:3000/ - The whole App example
- http://localhost:3000/demos - Other examples
- http://localhost:3000/?example=dbconnections - Access example one by one
